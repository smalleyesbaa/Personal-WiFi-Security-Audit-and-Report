# Personal-WiFi-Security-Audit-and-Report
Personal Wi-Fi Security Audit
This repository contains two PDF documents on a personal Wi-Fi security audit and a report performed on an Android mobile hotspot using Kali Linux. The project includes a step-by-step guide on how the security audit is conducted and a summarized audit report.

Contents
Personal Wi-Fi Step-by-Step Guide – This document shows commands used, reasons to use those commands, scanned images and general analysis

Final Audit Report – A precise and easy-to-read report that shows what was found, the strengths, risks and recommended actions.

Summary
The audit found strong security for the personal Wi-Fi with no known vulnerabilities and good access control.
